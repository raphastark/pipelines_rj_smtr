# Descrição do Projeto

Nome: Pipelines RJSMTR
Objetivo: Sistema de pipelines de dados para a Secretaria Municipal de Transportes do Rio de Janeiro

## Tecnologias

- Python
- SQL (dbt)
- Prefect (orquestração)
- BigQuery

## Estrutura

- pipelines/: Código Python para ETL e processamento de dados
- queries/: Modelos dbt para transformação de dados
- Organização por domínios de negócio (transporte, financeiro, etc.)
